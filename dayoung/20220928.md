### 애플리케이션 파일 만들기
```commandline
from flask import Flask
app = Flask(__name__) # 플라스크 애플리케이션 생성코드

@app.route('/')
def hello_pybo():
    return 'Hello, Pybo!'
```

### 플라스크 서버 실행하기
> 터미널에 __flask run__ 명령어 입력
> 주의사항!! 반드시 프로젝트 홈디렉터리에서 실행


### 디버깅 가능하도록 실행하기
> 터미널에 __set FLASK_DEBUG=true__ 입력
