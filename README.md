# SW-Mentoring

## TIL(Today I learend)

- 전공 관련이라면 무엇이든 어떤 주제든 상관 없다
- 최소 일주일의 3회 이상 Commit 한다.

### TIL 양식

- 공부한 내용만 명확하게 드러난다면 양식은 상관 없습니다.

#### Basic

```md
# [Title]

- Title은 오늘 배운 것의 주제를 짧게 적습니다

## [Findings]

- 공부한 것 위주로 정리합니다.
- 새로 알게된 내용
- 배운 것의 원리
- 참조한 레퍼런스 등등

## [Retrospective]

- 오늘 배우면서 느꼈던 점, 보완할 점을 기술합니다.
```
